Running The Application
  1. Connect a micro USB cable to the DEBUG port.
  2. Build and program the application using the MPLAB X IDE.
  3. Build the Application test_app using MPLAB X IDE
  4. Run the btl_host.py from command prompt to program the application binary. Refer to Bootloader Tools Help for setting up the host script

 

For SAM, PIC32 MZ and PIC32 MK Devices 

python <harmony3_path>\bootloader\tools\btl_host.py -v -i <COM PORT> -d <Device Name> -o <offset> -f <harmony3_path>\bootloader\apps\uart_bootloader\test_app\firmware\<Project Name>\dist\<Config Name>\production\<Project Name>.production.bin

 

For PIC32 MX Devices 

python <harmony3_path>\bootloader\tools\btl_host.py -v -i <COM PORT> -d <Device Name> -o <offset> -p <Sector Size> -f <harmony3_path>\bootloader\apps\uart_bootloader\test_app\firmware\<Project Name>\dist\<Config Name>\production\<Project Name>.production.bin

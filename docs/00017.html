Host Script 
Description 
Preconditions 
btl_host.py 

  • This host script should be used to communicate with the Bootloader running on the device via UART interface
  • It is a command line interface and implements the bootloader protocol required to communicate from host PC
  • If size of the input binary file is not aligned to device erase boundary it appends 0xFF to the binary to make it aligned and then sends the binary to device
  • It should be used with -s (swap banks) option when using the merged binary generated by btl_app_merge_bin.py for fail safe update and the offset going to device will be 0x0 as the binary has both bootloader and application

 

  • Compatible with Python 2.7.x
  • Requires pyserial package to communicate with device over UART
    • python -m pip install pyserial

 

btl_app_merge_bin.py 

  • This script should be used to merge the bootloader binary and application binary
  • It creates a merged binary output where bootloader is placed from start and the application will be placed at the offset passed as parameter
  • If the application offset is not equal to end of bootloader offset it fills the gap with 0xFF until the application offset
  • The merged binary will be used by btl_host.py as input for fail safe update

 

  • Compatible with Python 2.7.x

 

 

Basic Mode Usage Example

 

python <harmony3_path>\bootloader\tools\btl_host.py --help

 

 

 

python <harmony3_path>\bootloader\tools\btl_host.py -v -i COM18 -d same5x -o 0x2000 -f <harmony3_path>\bootloader\apps\uart_bootloader\test_app\firmware\sam_e54_xpro.X\dist\sam_e54_xpro\production\sam_e54_xpro.X.production.bin

 

 

Fail Safe Update Mode Usage Example

 

python <harmony3_path>\bootloader\tools\btl_app_merge_bin.py --help

 

 

 

python <harmony3_path>\bootloader\tools\btl_app_merge_bin.py -o 0x2000 -b <harmony3_path>\bootloader\apps\uart_fail_safe_bootloader\bootloader\firmware\sam_e54_xpro.X\dist\sam_e54_xpro\production\sam_e54_xpro.X.production.bin -a <harmony3_path>\bootloader\apps\uart_fail_safe_bootloader\test_app\firmware\sam_e54_xpro.X\dist\sam_e54_xpro\production\sam_e54_xpro.X.production.bin

python <harmony3_path>\bootloader\tools\btl_host.py -v -s -i COM18 -d same5x -f btl_app_merged.bin

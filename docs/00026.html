Hardware Setup
  1. Project pic32mk_gpl_curiosity_pro.X
    • Hardware Used
      • PIC32MK GPL Curiosity Pro Development Board
    • Hardware Setup
      • Connect a micro USB cable to the USB to UART port J400. This will enumerate as a COM port on the PC.
      • Connect a micro USB cable to the USB DEBUG port J500

 

  1. Project pic32mk_mcj_curiosity_pro.X
    • Hardware Used
      • PIC32MK MCJ Curiosity Pro Development Board
      • USB UART click
    • Hardware Setup
      • Install an USB UART click board on to the mikroBUS Xplained Pro board
      • Connect the mikroBUS Xplained Pro board to the J501 header of the PIC32MK MCJ Curiosity Pro Development Board
      • Connect micro USB cable to the 'Debug USB' connector(J1) on the board to the computer.
      • Connect a mini USB cable from the USB UART click board to the computer.

 

  1. Project pic32mx_125_sk.X
    • Hardware Used
      • PIC32MX 1/2/5 Starter Kit
      • Starter Kit I/O Expansion Board
      • USB UART click
    • Hardware Setup
      • Mount starter kit on the J1 connector of I/O Expansion Board
      • Connect TX pin of USB UART click to U2RX pin of J11 header on Starter Kit I/O Expansion Board
      • Connect RX pin of USB UART click to U2TX pin of J11 header on Starter Kit I/O Expansion Board
      • Connect GND pin of USB UART click to GND pin of J11 header on Starter Kit I/O Expansion Board
      • Connect mini USB cable to the 'Debug USB' connector(J3) on the board to the computer.
      • Connect a mini USB cable from the USB UART click board to the computer.

 

  1. Project pic32mx470_curiosity.X
    • Hardware Used
      • PIC32MX Curiosity Development Board
      • USB UART click
    • Hardware Setup
      • Connect mini USB cable to the 'Debug USB' connector(J3) on the board to the computer.
      • Install an USB UART click board on to the mikroBUS socket J5
      • Connect mini USB cable to the USB UART click board. This will enumerate the USB to UART port.
      • Note: Ensure that the series resistors on the mikroBUS headers are of value 0 Ohms.

 

  1. Project pic32mz_das_sk.X
    • Hardware Used:
      • PIC32MZ Embedded Graphics with Stacked DRAM (DA) Starter Kit (Crypto)
    • Hardware Setup:
      • Connect a micro USB cable to the UART-USB port J5
      • For programming, connect a micro USB cable to the USB Debug port J19

    • Hardware Setup:
      • Connect a mini USB cable to the UART-USB port J11
      • For programming, connect a mini USB cable to the USB Debug port J3
